# aulas-remotas

## Exercícios de Python

## Descrição

Este repositório contém dois exercícios básicos em Python para prática de:

* Recebimento de dados do usuário (`input`)
* Conversão de tipos (`int`)
* Estruturas condicionais (`if/else`)
* Loops (`for`)

Os exercícios são:

1. Verificador de Maioridade
2. Tabuada de um número

---

## Exercício 1 – Verificador de Maioridade

### Objetivo

* Praticar `input()`, conversão de tipos (`int()`) e estrutura condicional `if/else`.
* Determinar se o usuário é maior ou menor de idade com base na idade fornecida.

### Funcionalidades

* Solicita o **nome** e a **idade** do usuário.
* Verifica se a idade é **maior ou igual a 18**.
* Exibe uma mensagem usando o nome do usuário.

### Exemplo de execução

```
Qual é o seu nome? Ana
Qual é a sua idade? 20
Olá, Ana, você é maior de idade.
```

## Exercício 2 – Tabuada

### Objetivo

* Praticar `input()`, conversão de tipos (`int()`) e loops `for`.
* Gerar a tabuada de um número inteiro fornecido pelo usuário.

### Funcionalidades

* Solicita que o usuário digite um número inteiro.
* Calcula e exibe a tabuada de 1 a 10 desse número.
* Mostra os resultados formatados de forma clara.

### Exemplo de Saída
Se o usuário digitar o número 7, a saída no console deve ser exatamente assim:
Digite um número para ver a tabuada: 7

```
Tabuada do 7
7 x 1  =  7
7 x 2  =  14
7 x 3  =  21
7 x 4  =  28
7 x 5  =  35
7 x 6  =  42
7 x 7  =  49
7 x 8  =  56
7 x 9  =  63
7 x 10 =  70
```
